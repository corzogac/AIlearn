# Ai-LEARN Project Website

Main public website for the **Ai-LEARN: Environmental & Water Learning** Erasmus+ project.

## Project Info

- **Partners:** IHE Delft (NL), UPV/EHU (ES), TUC (GR)
- **Duration:** Oct 2025 – Sep 2028
- **Grant:** 2025-1-NL01-KA220-HED-000355215

## Purpose

This site serves as the primary dissemination channel for the project (WP4).  
It should include:

- Project overview and objectives
- Partner institutions and team
- News and events
- Learning resources and tools (links to 02 and 03)
- Publications and deliverables
- Contact information
